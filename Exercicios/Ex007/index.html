Principais Formatações

Negrito / Destaque

Para usar negrito em uma frase, basta usar a tag <b>

Já para usar um termo em destaque é necessário usar a tag <strong>

Itálico / Ênfase

Também é possível utilizar o itálico como formatação através da tag <i>

Para dar um valor semântico de ênfase, é só utilizar a tag <em>

Texto Marcado

A tag <mark> pode ser usada para marcar o texto.

Nesse novo mark está sendo usado as declarações do style na head do programa.

Texto grande e texto pequeno

Para criar um texto grande e para criar um texto pequeno use as tags <big> e <small>

Texto Deletado

Pode-se marcar um texto como deletado, para desconsiderar o que foi falado, através da tag <del>.

Texto Inserido / Sublinhado

Pode-se marcar um texto como inserido, para indicar que foi adicionado posteriormente, através da tag <ins>.

Existe a tag <u> que garante um sublinhado no texto

Texto Sobrescrito

Para inserir coisas do tipo x2 + 3, através da tag <sup>

Texto Subscrito

Para inserir coisas do tipo CO2, através da tag <sub>
